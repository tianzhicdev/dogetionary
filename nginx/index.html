Dogetionary API

AI-powered dictionary with spaced repetition learning

API Endpoints

GET /api/word?w={word}
Search for word definitions and pronunciations
POST /api/save
Save a word to your personal dictionary
GET /api/saved_words?user_id={uuid}
Get all saved words for a user
GET /api/saved_words/next_due?user_id={uuid}&limit=10
Get next words due for review
POST /api/reviews/submit
Submit a review response (SuperMemo SM-2 algorithm)
GET /api/reviews/stats?user_id={uuid}
Get review statistics and progress
GET /api/health
API health check
